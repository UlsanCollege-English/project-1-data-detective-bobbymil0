# Data Detective Project

## Problem
Analyze a text file and find the most frequent words.

## Features
- Load text from file
- Count word frequency
- Show top N words
- Bonus: most frequent word, total words, unique words

## How to run
python src/project.py

## How to test
python -m pytest

## Folder Structure
data/ - input files  
src/ - main code  
tests/ - test cases
